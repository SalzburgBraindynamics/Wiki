# MEG : Advanced Analyses - Tutorial Documentation

This section covers further methods and analyses that are useful when you want to explore your MEG data. It should not be understood as a complete (or gold-standard) instruction but as a general guideline for the informed researcher who wants to start with a new type of analysis but who knows his data well and what to expect in general.  This tutorial also provides you with a quick introduction into different MATLAB-based toolboxes that work nicely with data from very simple auditory stimulation such as (markov) tone sequences (e.g. MVPA-Light) or more complex continuous stimuli such as speech (e.g. MTRF).

Most of the examples might also include data preprocessing steps, as different preprocessing decisions (e.g. filter frequencies) might be advisable for different kinds of stimuli and/or analyses. But this tutorial does not replace a basic knowledge in how to read and preprocess your MEG data (but you can find the Basics Tutorial Documentation [here](https://im.sbg.ac.at/display/MEG/Basics+-+Tutorial+Documentation)).
