# MEG : 2a. Calculate source ingredients - surface source space

Now that we have our headmodel, we can calculate the following ingredients needed for source localization using minimum norm estimates (MNE) and a surface-based source space:

1. The noise covariance matrix obtained from an empty room measurement
1. The forward solution (aka leadfield)
1. The inverse operator

First, let's load/define some necessary stuff:

```py
# %% imports
import joblib
import mne
from os.path import join
from preproc_utils import preproc_data

# %% define stuff
# set paths
mri_path = '/mnt/obob/staff/preisinger/toolboxes/freesurfer/'
fs_path = join(mri_path, 'fsaverage')
src_file = join(fs_path, 'bem', 'fsaverage-ico-4-src.fif') # surface source space (icosahedron-4)
bem_file = join(fs_path, 'bem', 'fsaverage-5120-bem-sol-single-layer.fif')

# set filenames
fname_resting = "./data/subject01_resting_trans_sss.fif"
fname_trans = './data/subject01_headmodel.fif'  # the individual headmodel
fname_empty = "./data/empty_room.fif" # needed for the noise covariance matrix

# get info for MNE
info = mne.io.read_info(fname_resting, verbose=False)
info = mne.pick_info(info, mne.pick_types(info, meg=True))
```

Now calculate the noise covariance matrix, which is needed for the forward solution. To do so, we need to first preprocess the empty room measurement in the exact same way as the MEG data we want to use for source localization.

```py
# %% compute noise covariance matrix on empty room data
# do the same preprocessing as on the actual data, using a nice wrapper function by Fabi Schmidt
empty_room = preproc_data(fname_empty, max_filt=True, coord_frame='meg', notch=False,
                            apply_filter=True, hp_lp_freqs=(1, 10.), do_downsample=True,
                            resample_freq=100.)  

noise_cov = mne.compute_raw_covariance(empty_room, rank=None, picks='meg', method='auto')
true_rank = mne.compute_rank(noise_cov, info=empty_room.info) # inferring true rank
```

We can plot it and it should usually look something like this:

```py
# plot noise cov matrix
noise_cov.plot(info)
```

Great! Now we can compute the forward solution. Here we use the individual head model, a surface (icosahedral) source space and a boundary element method (BEM) model with one layer (i.e. the brain - EEG usually needs one with three layers, i.e. brain, skull and skin).

```py
# %% make forward solution
fwd = mne.make_forward_solution(info=info, trans=fname_trans, src=src_file, bem=bem_file)
```

Afterwards, we can make the inverse operator. Here we use fixed-orientiation dipoles and a depth weighting of 0.8, because MNE is biased towards superficial sources (between 0.6 and 0.8 is good, see also [here](https://doi.org/10.1016/j.neuroimage.2005.11.054)).

```py
# %% compute inverse operator for MNE (fixed orientation dipoles)
inv = mne.minimum_norm.make_inverse_operator(info, fwd, noise_cov, rank=true_rank, loose=0, fixed=True, depth=0.8)
```

Save everything, since it takes quite some time to calculate these things:

```py
# %% save everything 
data2save = {'inv': inv, 'fwd': fwd, 'noise_cov': noise_cov, 'empty_room_info': empty_room.info}
joblib.dump(data2save, './data/subject01_sourcestuff_surface.dat')
```
