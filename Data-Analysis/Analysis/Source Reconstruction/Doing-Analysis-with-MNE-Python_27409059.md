# MEG : Doing Analysis with MNE Python

[MNE Python](https://mne-tools.github.io/stable/index.html) is a package (i.e. a toolbox in Matlab-speak) for the [Python Programming Language](https://www.python.org/). It is actually quite similar to Matlab. The job it does is basically the same as FieldTrip. So if you have worked with Matlab and FieldTrip so far, the transition will not be that hard.

The following pages will show you how to install and use Python, MNE Python and everything else you need. If you have any comments, please let me know!
